Lab 8 - JavaScript, JSON and AJAX

In lab 3 we made a website which consisted of a homepage (index.html) and a projects page. Since then we have been adding projects to that page for each lab.

Well, here we go again. We are going to modify the projects page we created in that lab again, but this time in a different way.

In this lab,
  • You are to make a JSON file with a list of your project menu items

    I have provided a JSON template to get you started.

    I have also provided sample of Lab 7 from a previous year as an example of how to use AJAX to read a file

  • The file should contain the information necessary for you to be able to build the appropriate HTML for your projects page.
  • You are to write some JavaScript to read that file and create your projects menu dynamically for your website.

    Your javascript file must be external to your HTML file.

    Your jQuery must be served from a CDN of your choosing

  • when the projects page is loaded, it should read the json file and build your site dynamically
  • Add some jQuery or jQueryUI to Jazz-up your website

Before you begin,

  • You are to work in your iit repos for this lab
  • Create an issue in your GitHub issues describing what you need to do for lab 8
  • Create a new branch in your repo named lab8

When you are done,

  • Commit your updated iit folder to your GitHub repo
  • Merge your branch back into main on GitHub
  • Close out your issue on GitHub
  • Pull your changes down to your website and verify that all is working, then stop your server
  • Do not forget your README file, and make sure to include you GitHub and Azure links
  • Post a link to your repo on LMS

You will be graded according to the following criteria: Lab 8 Rubric
